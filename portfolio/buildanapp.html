Build An App

  • Projects
  • Flow 3
  • Build An App

This assigment asked of us that we created an app that would utilize one or more of FN Global Goals, in collaboration or for an already existing company.


The requirements for the assignment was as below stated:


  • The app most be developed both for a mobile aswell for a tablet
  • It most contain atleast three menu items
  • The contents most be sufficient enough to complete a user test

The idea for my app had starting point in a type of payment app, in which the user would have the option to donate a set amount of money to clean water (Global goal 6). In conclusion i ended up with the idea of an app that would let the users gift money while still keeping it personal and also having the same donation option. I then added more to the idea, by also making it a type of game where your progress/level would be measured in the amount of money donated. By earning more levels the user would then be unlocking more and more customization options to there gift.


When i was building this app i started off by laying the ground work of defining the target group, and in such creating the personas. With the target group specified i could now start on creating wireframes, mockups etc.
